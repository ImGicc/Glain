Gtopo
=====
ImGicc Global Token Pool for AI/ML

Files
=====
There's only 1 file named `tokens.js` for syllables of all languages and symbols.
This file is **append-only** and the indexes of tokens always stay the same
for added tokens.

More Tokens
===========
Please go [Issues](https://github.com/imgicc/gtopo/issues) tab to submit text files 
to have the tokens added.
